RAFIX STUDIO — ASSETS FOLDER
==============================

Place your files here:

📸 PHOTO
  - Save your photo as: rafid-photo.jpg (or any name)
  - Then set the path in Admin → Brand & Hero → Photo Path
  - Recommended size: 600x800px or similar portrait ratio

📄 RESUME
  - Save your resume PDF as: Rafid_Resume_2026.pdf (or any name)
  - Match the name in Admin → Brand & Hero → Resume File Name

🖼 PROJECT IMAGES
  - Create subfolders for each project
  - Example: assets/projects/brand-identity-01/cover.jpg
  - Then set paths in Admin → Projects → Cover Image Path

Supported image formats: JPG, PNG, WebP
