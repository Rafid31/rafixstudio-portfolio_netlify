Getting Started
1. Add your photo
Go to Brand & Hero → paste the path to your photo (e.g., assets/rafid-photo.jpg). Place the file in the rafix-studio/assets/ folder.
2. Upload your resume PDF
Go to Brand & Hero → Hero Photo & Resume and click "Upload PDF". Your resume is stored in the browser and served directly when visitors click Download CV.
3. Add your projects
Go to Projects → click Add New Project → fill in details and save.
4. Deploy your site
Go to Deploy Guide for free hosting options on Netlify or GitHub Pages.

🚀 Deploy Your Portfolio
Get your portfolio live on the internet — free, fast, and no coding required.

Option 1: Netlify Drop (Easiest — Recommended)

No account needed. Just drag and drop your entire rafix-studio folder.

  1. Go to netlify.com/drop
  2. Drag your rafix-studio folder onto the page
  3. Your site is live instantly at a random URL
  4. Create a free Netlify account to get a custom URL like rafixstudio.netlify.app
🌐 netlify.com/drop

Option 2: GitHub Pages (Free Custom Domain)

Free hosting on GitHub with a username.github.io URL or your own custom domain.

  1. Create a free GitHub account at github.com
  2. Create a repository named rafidkhan.github.io
  3. Upload all files from your rafix-studio folder
  4. Enable GitHub Pages in Settings → Pages → Deploy from branch: main
  5. Your site is live at rafidkhan.github.io
🌐 github.com/new

Option 3: Vercel (Best Performance)

Connect your GitHub repo to Vercel for automatic deployments on every update.

🌐 vercel.com/import
Custom Domain Setup

After deploying, you can connect your own domain (like rafixstudio.com) from your hosting platform's settings. Buy a domain from Namecheap or GoDaddy, then follow the DNS configuration instructions in your Netlify/Vercel dashboard.

Before You Deploy — Checklist
Export & Import Data
Back up your CMS data or transfer it to another device.
Export Data

Export all your site data as a JSON file. Save this as a backup, or use it to update the data.js file directly for permanent deployment.

Import Data

Restore data from a previously exported JSON file. This will overwrite current data.

Reset to Defaults

Clear all saved changes and restore the original data from data.js.
